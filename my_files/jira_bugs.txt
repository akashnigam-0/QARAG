BUG-101: Job failing for Kubernetes Spark image runtime - due to provided code

Status: Resolved | Priority: High | Date: 2024-11-10

Description:

Job execution is failing on the QA environment when using the Kubernetes Spark image runtime. The job does not complete successfully and ends in a failed state during Spark processing.

GCP:

[Root Cause]: Exception: Spark Code Process failed with error code [1] : 'GooglePyPackageInstaller' object has no attribute 'node_ip_addresses' 2026-01-02 08:12:31.823 WARN test--spark-spark--proc-driver --- [ool-21-thread-1] .s.s.c.k.ExecutorPodsWatchSnapshotSource : Kubernetes client has been closed.

AWS:

[Root Cause]: Exception: Spark Code Process failed with error code [1] : 'AmazonClusterPyPackageInstaller' object has no attribute 'node_ip_addresses' 2026-01-02 08:52:38.315 WARN test--spark-spark--proc-driver --- [ool-22-thread-2] .s.s.c.k.ExecutorPodsWatchSnapshotSource : Kubernetes client has been closed.

Steps to Reproduce:

Login to QA environment

Create or open a job configured with Kubernetes Spark image runtime

Execute the job

Monitor job execution and logs

Actual Result:
Job execution fails on QA when running with Kubernetes Spark image runtime.

Expected Result:
User should be able to execute jobs successfully on QA using Kubernetes Spark image runtime without failures.
